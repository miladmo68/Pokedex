## ​ Getting Started

```bash
# 1) Clone the repository
git clone https://github.com/miladmo68/Pokedex.git
cd Pokedex

# 2) Install dependencies
npm install

# 3) Start the development server
npm start

# 4) Build the app for production
npm run build
